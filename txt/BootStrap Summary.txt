Video 1
Install bootstrap

1- CMD 

2- npm

3- Download

    1- Download bootstrap v4.6 (Compiled CSS and JS bootstrap):
        https://getbootstrap.com/docs/4.6/getting-started/download/

    2- Download jQuery v3.5.1: 
        https://blog.jquery.com/2020/05/04/jquery-3-5-1-released-fixing-a-regression/

    3- index.html: 
        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Bootstrap 4</title>
            <link rel="stylesheet" href="css/bootstrap.min.css" />
        </head>
        <body>
            <div class="container">
                <h1>Hello Bootstrap 4</h1>
            </div>

            <script src="js/jquery-3.5.1.min.js"></script>
            <script src="js/bootstrap.min.js"></script>
        </body>
        </html>
